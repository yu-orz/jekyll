なぜか git commit してコミットメッセージ入力した後に以下エラーが出る場合。

error: There was a problem with the editor 'vi'.
Please supply the message using either -m or -F option.

とか出る場合、.zshrcなり.bashrcなりに以下を追加する。

export GIT_EDITOR=`which vim`
