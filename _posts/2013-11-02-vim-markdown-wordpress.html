Vim で Wordpress の記事を書きたいなーついでに Markdown で書きたいなーということでやってみました。

Vunble のインストール

数多くのプラグインをインストールするため、 vundle を導入します。

[bash]
$ mkdir ~/.vim
$ cd ~/.vim
$ git clone http://github.com/gmarik/vundle.git
[/bash]

.vimrc に数行追加します。

[bash]
$ vim ~/.vimrc
[/bash]

filetype off
set rtp+=~/.vim/vundle.git/
call vundle#rc()

" ここからリポジトリ
Bundle "rails.vim"
Bundle "grep.vim"
Bundle "The-NERD-tree"
Bundle "syntastic"
Bundle 'tpope/vim-markdown'
Bundle 'tyru/open-browser.vim'
Bundle 'thinca/vim-quickrun'
Bundle "vim-scripts/VimRepress"
Bundle 'Shougo/neocomplcache'
Bundle "tpope/vim-fugitive"
Bundle "kannokanno/previm"
Bundle "git://git.wincent.com/command-t.git"

filetype plugin indent on

"filetype
autocmd BufNewFile,BufRead *.{md,mdwn,mkd,mkdn,mark*} set filetype=markdown

上記を追加して vim を開き :BundleInstall を実行すればOK

Markdownの準備

[bash]
$ sudo easy_install Markdown
[/bash]

Wordpressへの投稿

.vimpressrc に以下を追加します。

[Blog0]  
blog_url = BlogのURL
username = ユーザ名
password = パスワード

以上で準備完了です。

使い方

記事一覧

:BlogList

新規追加

:BlogNew

投稿

:BlogSave public    (公開)
:BlogSave           (Draft)

プレビュー

:BlogPreview

難しそうに見えないけど、最初は手間でした。
