VAIOのtypeP(VGN-P70H)
ちなみにワンセグモデル。

もワンセグとかどうでもいいし、デフォルトのVistaが死ぬほど重たくてどうにかなりそうなので
ubuntu 9.10 入れてみようと思います。



前提

リカバリパーティションは残す
ワンセグは使わない
ついでにWindows 残す

あくまで自分の環境なので、ご自身の環境にあわせてください。

Windows消すなりリカバリ消すなり。

認識するデバイス

グラフィック： ドライバ入れればファンクションキーでの輝度調整も出来る
サウンド： 問題なし
無線LAN： 問題なし
カメラ： 問題なし

インストール

普通にインストールします。特記することはとくにないかも。

初期設定

まずまっさきにアップデートします。
ただしカーネルだけはアップデートしないでください。

# apt-get update
# apt-get upgrade

次にグラフィックドライバを落とす。
# wget http://gma500re.altervista.org/scripts/poulsbo_ppa.sh

インストール
# sh poulsbo_ppa.sh

終わったら grub の設定を変えます。
# vi /etc/default/grub

GRUB_CMDLINE_LINUX_DEFAULT="quiet"
GRUB_CMDLINE_LINUX="mem=2000mb"

編集したら適用する
# update-grub

再起動
# reboot

これで終わり。
音も出るし輝度調整もできます。

また続くかも。
