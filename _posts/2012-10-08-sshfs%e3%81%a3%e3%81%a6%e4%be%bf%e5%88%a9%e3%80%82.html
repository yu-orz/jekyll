レンタルサーバを別に借りたので、自宅サーバとレンタルサーバでファイルの受け渡しが簡単にできないかなーと。 そこで思いついたのがsshfsです。 簡単に言うと、ssh経由でリモートサーバのディレクトリをマウント出来るファイルシステムがこれ。

以下、CentOS6でのインストール方法と使い方です。

[bash]
# yum -y install sshfs
[/bash]
インストール自体はこれで完了で、使い方です。
[bash]
# sshfs user@server:/path/to/dir /mnt/dir # マウント
# fusermount -u /mnt/dir                # アンマウント
[/bash]

これでNFSのように使用できます。
注意点としては、sshfsでマウントしたユーザにしかこのディレクトリは見えません。
設定かマウントオプションで変えられるかもしれませんが。。
